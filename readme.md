# math-sum

> Sum numbers


## Install

```
$ npm install --save math-sum
```


## Usage

```js
var mathSum = require('math-sum');

mathSum(5, 5);
//=> 10

var arr = [1, 2, 3, 4];
mathSum(arr);
//=> 10
```
